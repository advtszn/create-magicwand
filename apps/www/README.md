# Magicwand CLI Website

Marketing site for `create-magicwand`, the CLI for scaffolding pragmatic layered DDD backends.

## Local Development

```bash
bun install
bun dev
```

## Notes

- The current site keeps the original 4-step landing page structure from the cloned template.
- Copy, metadata, and calls to action are aligned to Magicwand CLI.
- The page content mirrors the current v1 backend scaffold flow: generate, customize, inspect, and run.
